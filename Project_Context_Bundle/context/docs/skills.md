# NetListLab Roles & Skills Reference

## Target User Personas
NetListLab caters to a diverse range of "builders". When designing onboarding flows, tag inputs, and profile cards, keep these core roles in mind:

1. **Software Engineers / Developers**
   * *Skills:* React, TypeScript, Python, Rust, Go, Node.js, System Design.
   * *Interests:* Open source, architecture, performance, APIs.
2. **Designers (UI/UX, Product)**
   * *Skills:* Figma, User Research, Prototyping, Interaction Design.
   * *Interests:* Design systems, typography, glassmorphism, accessibility.
3. **Researchers / Scientists**
   * *Skills:* Machine Learning, PyTorch, Data Analysis, Academic Writing.
   * *Interests:* Generative AI, algorithms, publishing, experimentation.
4. **Hardware Engineers / Makers**
   * *Skills:* CAD, PCB Design, Arduino, 3D Printing, Robotics.
   * *Interests:* Prototyping, IoT, physical computing.
5. **Students**
   * *Skills:* Various entry-level skills, fast learning.
   * *Interests:* Building projects, finding mentors, securing internships.

## Codebase Skill Requirements
If you are contributing to this repository, the following technical skills are required:
*   **Expert React & Hooks:** Deep understanding of component lifecycle, `useCallback`, `useMemo`, and custom hook creation.
*   **TypeScript Mastery:** Strict typing, interfaces, generic types.
*   **CSS Architecture:** Understanding of CSS Custom Properties (Variables), scoping, flexbox/grid layouts, and advanced selectors.
*   **Animation Engineering:** Proficiency with `framer-motion` variants, layout animations, and spring physics balancing.
